WebAssembly (аббревиатура Wasm) это набор бинарных инструкций для стэковой машины. Wasm создан как портативная среда для языков програмированния, позволяющая размещать себя на клиенте или сервере.

Зачем?



все яндекс игры написаные на unity, все сложные приложения по типу Figma работающие в браузере используют wasm для вычислений на стороне клиента. также wasm применяют на стороне сервера для запуска онлайн компиляторов (см. WasmEdge) можно эмулировать DOS/NES/C64 и многие другие для игр и софта.
